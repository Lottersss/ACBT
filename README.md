# Logic Simulator

**Logic Simulator** — это визуальный симулятор логических схем, написанный на Python с использованием Tkinter. Проект позволяет создавать схемы с логическими элементами (AND, OR, XOR, NOT, NAND, NOR, XNOR), входами и выходами, соединять их проводами и автоматически вычислять таблицу истинности.

---

## Основные возможности

- **Добавление логических элементов**: AND, OR, XOR, NOT, NAND, NOR, XNOR
- **Входы и выходы**: INPUT и OUTPUT
- **Соединение проводами** с отображением текущего состояния (0/1)
- **Перетаскивание гейтов** мышью
- **Вращение гейтов** на 0°, 90°, 180°, 270°
- **Плавные провода**, автоматически подстраивающиеся под ориентацию гейтов
- **Таблица истинности** для всех входов и выходов
- **Сохранение и загрузка схем** в формате JSON
- **Масштабирование и смена темы** (темная/светлая)
